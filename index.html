Self-service tally sheet


Manage your kiosk with ease.

Touch-friendly

Works without JavaScript

Features

Statistics

Stay on top of your organization's finances: the metrics page shows the sum of all balances, transaction volume and top articles — and every user gets personal statistics.

Database agnostic

strichliste runs on the database you already have: SQLite for a single kiosk, PostgreSQL or MariaDB for busy installs. Switching is a single setting — use what you like!

REST-API

Write your own client for our easy-to-use REST API — with interactive documentation built right into your installation at /api/doc.

Web-based

Being a web application, strichliste can be accessed on any device that runs a web browser — and the UI works even with JavaScript disabled.
